# 🎓 College Event Management System

A full-stack web application designed to simplify the management of college events. This platform allows administrators to create and manage events, while students can register, receive updates, and participate easily.

---

## 🚀 Features

### 👨‍🎓 Student Features

* Register and create an account
* Browse available events
* Register for events
* Receive notifications and updates

### 🛠️ Admin Features

* Create and manage events
* View registered participants
* Monitor event activity
* Manage event details

### 🤖 AI Integration

* Uses Gemini API for intelligent features
* Smart suggestions and automation

---

## 🧰 Tech Stack

### Frontend

* React.js / Next.js
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js
* TypeScript

### Database

* SQLite / MongoDB (based on setup)

### Deployment

*  Render

---


---

## 🌐 Deployment

### Frontend (Netlify)

* Connect GitHub repository
* Add environment variables
* Deploy

### Backend (Render)

* Create Web Service
* Add environment variables
* Start command:

```bash
npm run dev
```

---

## 🔐 Environment Variables

| Variable       | Description                 |
| -------------- | --------------------------- |
| GEMINI_API_KEY | API key for AI features     |
| APP_URL        | Base URL of the application |
